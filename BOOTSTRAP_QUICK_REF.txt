┌─ BOOTSTRAP SEQUENCE ─────────────────────────────────────────────────────────┐
│                                                                               │
│  1. Docker Container Start                                                   │
│     ↓                                                                         │
│  2. entrypoint.sh (shell script)                                             │
│     • Connects to PostgreSQL                                                 │
│     • Acquires advisory lock: pg_advisory_lock(424242)                       │
│     • Creates 16 tables (IF NOT EXISTS)                                      │
│     • Handles backward-compat migrations                                     │
│     • Releases advisory lock                                                 │
│     ↓                                                                         │
│  3. Node.js server starts: node server.js                                    │
│     ↓                                                                         │
│  4. Next.js initializes instrumentation.ts                                   │
│     • Checks NEXT_RUNTIME environment variable                               │
│     • Imports instrumentation-node.ts if Node.js runtime                     │
│     ↓                                                                         │
│  5. instrumentation-node.ts (HMR-safe global guard)                          │
│     • Schedules quota alert checker (+30s delay)                             │
│     • Schedules provider resync (+15s delay)                                 │
│     • Uses recursive setTimeout (DB-driven intervals)                        │
│     ↓                                                                         │
│  6. Server ready at http://localhost:3000                                    │
│                                                                               │
