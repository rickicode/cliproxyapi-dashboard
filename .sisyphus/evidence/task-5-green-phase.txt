TASK 5 — GREEN PHASE — Fix buildSlimConfig() to stop skipping agents when pickBestModel() returns null

COMPLETED: 2026-04-07

## Summary
Fixed the bug in buildSlimConfig() that was skipping agents when no models matched a tier.
Previously, when availableModels was empty, all 6 agents would be skipped via `continue` at line 50,
causing the function to return null. Now agents are still included with a placeholder model.

## Change
File: dashboard/src/lib/config-generators/oh-my-opencode-slim.ts (lines 48-52)

Before:
```typescript
} else {
  const model = pickBestModel(availableModels, role.tier);
  if (!model) continue;  // BUG: skips agent entirely
  entry = { model: `cliproxyapi/${model}` };
}
```

After:
```typescript
} else {
  const model = pickBestModel(availableModels, role.tier);
  if (!model) {
    // No model available for this tier — use placeholder to keep agent in config
    entry = { model: `cliproxyapi/unresolved-tier-${role.tier}` };
  } else {
    entry = { model: `cliproxyapi/${model}` };
  }
}
```
